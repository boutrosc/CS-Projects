Installation: 
Open command prompt
Navigate to folder containing program files
Enter 'javac Driver.java' to comile the program
Enter 'java Driver' to run the program

Usage: Upon running program, you will be asked for the number of races to run and the numbers of cars to race. An animation will then be displayed, showing the races. After completion of all races, a summary will printed to the console show the number of wins for each car and the slowest overall car based on distance from finish line (not the number of races lost).
