**********
HOW TO RUN
**********

1. Open a terminal

2. Change directory to file location

3. Enter "gcc filesystem.c" into the terminal



**********
HOW TO USE
**********


Upon startup, you will see a "$ " prompt, you 
may enter 
any of the following commands list below:



ls                    ** lists all files and directories in the current directory, indicating which (file or directory) it is

mkdir <directoryName> ** creates a new directory if it does not already exist

cd <directoryName>    ** changes into specified directory if it exists

cd ..                 ** changes to the parent directory

pwd                   ** specifies the current directory:yourname/root/nextdir/etc/

addf <filename>       ** adds a file to the current directory

mv <fileName1> <fileName2> ** change the name of the file or directory to the new name

rm <fileName>         ** locate and remove the file or directory

whereis <fileName>    ** show path to ALL occurrences of file or directory if it exists

bye                   ** ends the session



***************PLEASE NOTE*************************

*SPACES ARE NOT ALLOWED IN FILE OR DIRECTORY NAMES*

*DO NOT USE MORE PARAMETERS THAN WHAT IS NEEDED****

*DO NOT ADD A SPACE AFTER THE LAST PARAMETER*******
