Author: Charbel Boutros

Name: Thread Synchronization

Installation: 
Open command prompt
Navigate to folder containing program files (Makefile, lizard.c, README)
Enter 'make' to compile the program
To run the program with debugging mode enabled you must use '-d' as a parameter
Enter './lizard' or './lizard -d' to run the program normally or with debugging mode enabled, respectively.
After usage enter 'make clean' to remove program


Purpose: The purpose of the program is to implement thread synchronization through the use of locks and semaphores

Usage: After executing the program with debugging mode disabled, nothing will print other than error messages (if applicable) and the program will terminate when done executing. After executing the program with debugging mode enabled, you will be displayed will status updates of all running threads and the program will terminate when done executing.
